# Projeto — Tabela Hash (simples)

Instruções mínimas para compilar e executar:

1. Abra o terminal e vá para o diretório do projeto:
   cd /workspaces/estrutura-de-dados-projeto

2. Compile com gcc:
   gcc -Wall -o auth tabelaHASH.c

3. Execute o programa:
   ./auth

Notas:
- É necessário ter o compilador gcc instalado.
